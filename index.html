I improve software safety and security
-- ensuring that software does what it is supposed to (safety)
and does not allow what it is not supposed to (security) --
  • by working with clients to review and improve their software, using modern program analysis tools to reason about and detect errors in software
  • and my own research and publication in software specification, verification, and correctness.

Current Professional Activities

  • Consulting to improve or validate software (2017 - present)
  • Visiting scientist at the LSL lab at CEA (near Paris, France), contributing to the ACSL++ extension of ACSL to C++ and the frama-clang plugin to Frama-C to support verification of C++ programs with specifications written in ACSL++
  • Significant contributor to JML (the Java Modeling Language) for writing behavioral specifications for Java
  • Principal developer of the open source tool OpenJML to verify Java programs against JML specifications.
  • Contributor to the ACSL specification language for C programs
  • Contributor to SMT-LIB, a standard language for interacting with SMT solvers
